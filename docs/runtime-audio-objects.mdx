# Getting the `AudioComponent` and `SoundWave` object and changing `SourceEffectChain`, Volume, Attenuation at runtime

The audio component and sound wave used to play voice can be retrieved at runtime.

You can use all available default Unreal Functions for `UAudioComponent` and `USoundWave`. For example, `SetVolumeMultiplier`.
